
# CodeLive - Online IDE
Simple online editor that lets you write and run HTML, CSS, JavaScript code in a snap. No installation, no setup, just code and go.

## Features

- HTML, CSS, JS Support
- Live and realtime previews
- Auto-saving code to Local Storage


## Tech Stack

**Client:** React, FontAwesome, react-codemirror




## Run Locally

Clone the project

```bash
  git clone https://github.com/Battery9/CodeLive
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
